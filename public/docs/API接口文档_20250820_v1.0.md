## 2. 提交数据表和数据库内容

**接口名称**: 提交数据表和数据库内容

**请求方法**: POST

**请求地址**: https://<ip>:port/prod-api/system/api/v1/tasks

**请求数据类型**: application/json

**请求参数**: 

#### Header参数
|参数名|类型|必填|说明|
|---|---|---|---|
| Authorization | string | 否 | ApiKey值，例如：Ql4kVpNW2Zj3UF44MC7HSimNMnVVC285 |

#### Body参数
|参数名|类型|必填|说明|
|---|---|---|---|
| taskId | string  | 是   | 任务ID |
| dataSourceName | string   | 是   | 数据源名称   |
| sourceBusinessSystem | string   | 是   | 来源业务系统 |
| databaseList | [object] | 是   | 数据库列表   |
| databaseName | string   | 是   | 数据库名     |
| tableList | [object] | 是   | 数据表列表   |
| tableName | string   | 是   | 表名         |
| tableRemark | string   | 是   | 表注释       |
| tableFieldList | [object] | 是   | 字段列表     |
| tableFieldName | string   | 是   | 字段名       |
| tableFieldRemark | string   | 是   | 字段注释     |
| tableFieldDataList | [string] | 是   | 样本内容     |


**响应示例**:
```json
{
  "msg": "任务已接收",
  "code": 200,
  "data": "taskID:18"
}
```

**响应数据结构**:
| 名称   | 类型   | 必选 | 约束 | 中文名   | 说明 |
|---|---|---|---|---|---|
| » msg  | string | true | none | 提示信息 | none |
| » code | string | true | none | 状态码   | none |
| » data | object | true | none | 数据值   | none |

| » taskId        | string | 是   | 任务ID                      |
